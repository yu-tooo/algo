# insertion sort
前から比較していくが、入れ替えが発生した場合、その入れ替えた数が正しい位置に来るまで入れ替えを続けていく。

例
[1, 3, 9, 2]
9と2を入れ替えるとした場合、
[1, 3, , 9] temp = 2
入れ替えた数（この場合2）の本来の位置に来るまでさかのぼって入れ替えを続けていく
[1, 2, 3, 9]
